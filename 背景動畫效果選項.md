# 背景動畫效果選項

## 當前已有的動畫效果

### 1. **漸變移動 (Gradient Shift)**
- **效果**：漸變背景位置平滑移動
- **適用**：純色漸變背景
- **代碼**：`@keyframes gradientShift`
- **狀態**：已定義但未使用

### 2. **網格移動 (Grid Move)**
- **效果**：網格圖案緩慢移動
- **適用**：網格覆蓋層
- **代碼**：`@keyframes gridMove`
- **狀態**：已定義但未使用

## 可添加的背景動畫效果

### 3. **背景圖片緩慢縮放 (Ken Burns Effect)**
```css
@keyframes backgroundZoom {
  0% {
    transform: scale(1);
  }
  100% {
    transform: scale(1.1);
  }
}
```
- **效果**：背景圖片緩慢放大，創造深度感
- **適用**：背景圖片
- **性能**：良好

### 4. **背景圖片平移 (Parallax Pan)**
```css
@keyframes backgroundPan {
  0% {
    transform: translateX(0) translateY(0);
  }
  50% {
    transform: translateX(-5%) translateY(-3%);
  }
  100% {
    transform: translateX(0) translateY(0);
  }
}
```
- **效果**：背景圖片緩慢平移
- **適用**：背景圖片
- **性能**：良好

### 5. **覆蓋層脈動 (Overlay Pulse)**
```css
@keyframes overlayPulse {
  0%, 100% {
    opacity: 0.6;
  }
  50% {
    opacity: 0.8;
  }
}
```
- **效果**：覆蓋層透明度緩慢變化
- **適用**：顏色覆蓋層
- **性能**：優秀

### 6. **漸變旋轉 (Gradient Rotate)**
```css
@keyframes gradientRotate {
  0% {
    background-position: 0% 50%;
  }
  50% {
    background-position: 100% 50%;
  }
  100% {
    background-position: 0% 50%;
  }
}
```
- **效果**：漸變方向緩慢旋轉
- **適用**：漸變背景
- **性能**：良好

### 7. **粒子漂浮 (Particle Float)**
```css
@keyframes particleFloat {
  0%, 100% {
    transform: translateY(0) translateX(0);
    opacity: 0.3;
  }
  50% {
    transform: translateY(-20px) translateX(10px);
    opacity: 0.6;
  }
}
```
- **效果**：小點或粒子緩慢漂浮
- **適用**：裝飾性元素
- **性能**：中等（需多個元素）

### 8. **光暈擴散 (Glow Spread)**
```css
@keyframes glowSpread {
  0% {
    box-shadow: 0 0 20px rgba(6, 182, 212, 0.3);
  }
  50% {
    box-shadow: 0 0 40px rgba(6, 182, 212, 0.5);
  }
  100% {
    box-shadow: 0 0 20px rgba(6, 182, 212, 0.3);
  }
}
```
- **效果**：光暈效果擴散和收縮
- **適用**：重點區域
- **性能**：良好

### 9. **網格閃爍 (Grid Flicker)**
```css
@keyframes gridFlicker {
  0%, 100% {
    opacity: 0.15;
  }
  50% {
    opacity: 0.25;
  }
}
```
- **效果**：網格線透明度緩慢變化
- **適用**：網格覆蓋層
- **性能**：優秀

### 10. **視差滾動 (Parallax Scroll)**
- **效果**：背景移動速度與頁面滾動不同
- **適用**：背景圖片
- **實現**：需要 JavaScript
- **性能**：中等

## 推薦組合

### 輕量級組合（性能最佳）
1. **覆蓋層脈動** + **網格閃爍**
   - 性能：優秀
   - 視覺效果：微妙但有效

### 中等效果組合
2. **背景圖片緩慢縮放** + **覆蓋層脈動**
   - 性能：良好
   - 視覺效果：有深度感

### 豐富效果組合
3. **背景圖片平移** + **漸變旋轉** + **網格移動**
   - 性能：中等
   - 視覺效果：動態豐富

## 注意事項

⚠️ **性能考量**：
- 避免同時使用太多動畫
- 使用 `will-change` 屬性優化
- 考慮使用 `transform` 而非 `position` 屬性
- 測試移動設備性能

⚠️ **可訪問性**：
- 提供 `prefers-reduced-motion` 媒體查詢支持
- 避免過度動畫影響閱讀

## 實施建議

1. **輕微動畫**：覆蓋層脈動 + 網格閃爍（推薦）
2. **中等動畫**：背景圖片緩慢縮放
3. **豐富動畫**：背景圖片平移 + 漸變旋轉
